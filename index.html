Function Analyzer

Example: f(x) = (x^2 - 4) / (x - 2)

How to enter functions correctly:

  • Use x as the variable.
  • Write powers like x^2, x^3, etc.
  • Group expressions using parentheses: (x^2 - 4)/(x - 2).
  • Multiplication must be explicit: 2*x instead of 2x.
  • Use sqrt(x) for square roots and ln(x) for natural log.
